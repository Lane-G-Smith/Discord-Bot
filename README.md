# Dick-bot

## Building and running on localhost

## install dependencies:
```sh
npm install
```
## create a production build:

```sh
npm run build-prod
```
## create a development build:
```sh
npm run build-dev
```
## run
```sh
node dist/bundle.js
```
## test
To run unit tests:

```sh
npm test
```
